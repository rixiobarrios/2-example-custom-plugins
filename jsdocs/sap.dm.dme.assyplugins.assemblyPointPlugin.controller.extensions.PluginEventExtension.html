PluginEventExtension

PluginEventExtension

The Assemble Components PluginEventExtension class allows modification of core plugin behaviour when the Assemble Components plugin is running. Custom extensions can be created to override or modify core behaviour. The following POD Plugin event functions can be overridden by custom extensions:

   onWorklistSelectEvent()
   onInputChangeEvent()
   onOperationChangeEvent()
   onComponentChangeEvent()
   onComponentRemoveEvent()
   onAutoAssemblySuccess()
   onAutoAssemblyError()
   generateAssemblyDataFields()
   onAddPress()
   postAddRequest()
   onSuccessfulAdd()
   onFocusToDataField()
   onFocusToFindComponent()
   onAddError()
   onSkip()
   onClose()

The following functions are available to extensions: 

   getAssemblyDataModel()
   getAssemblyDataFieldsForm()
