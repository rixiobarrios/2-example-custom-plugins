    ActionAssignmentHelper

    ActionAssignmentHelper

    sap.dm.dme.podfoundation.control.ActionAssignmentHelper will display a dialog to display Action Assignments

    Constructor

    new ActionAssignmentHelper(oPropertyEditoropt, sButtonTypeopt, sAssignmentIIdopt)

    Constructor for class that will display a dialog for assigning actions from a sap.dm.dme.podfoundation.control.PropertyEditor. When the dialog closes, the sap.dm.dme.podfoundation.control.PropertyEditor#updateAssignedActions function will be called passing the assigned actions and assignment ID that was passed to the constructor.

    Parameters:
    Name Type Attributes Description
    oPropertyEditor object <optional>

    Property Editor launching the Action Assignment dialog
    sButtonType string <optional>

    Type of button ("ACTION_BUTTON" or "MENU_BUTTON")
    sAssignmentIId string <optional>

    Identifier for this assignment

    Extends

    • sap.ui.base.Object

    Methods

    showActionAssignmentDialog()

    Displays the dialog for assigning Actions to a Menu or Action button. When the dialog closes, the sap.dm.dme.podfoundation.control.PropertyEditor#updateAssignedActions function will be called passing the list of assigned actions and the assignment ID that was passed to the constructor for this class.

    The PropertyEditor that invokes this method can implement the following functions: 

    getMenuItemLabelList(sAssignmentIId) should return an object containing a list of i18n names
to display as suggestions for menu names (Menu Button type only).  This is optional.

    getActionAssignmentDialogTitle(sAssignmentIId) should return the title for the dialog

    showAddRemoveActionAssignmentDialog()

    Displays the dialog for assigning Actions to a Menu or Action button. When the dialog closes, the sap.dm.dme.podfoundation.control.PropertyEditor#updateAssignedActions function will be called passing the list of assigned actions and the assignment ID that was passed to the constructor for this class.

    The PropertyEditor that invokes this method can implement the following functions: 

    getMenuItemLabelList(sAssignmentIId) should return an object containing a list of i18n names
to display as suggestions for menu names (Menu Button type only).  This is optional.

    getActionAssignmentDialogTitle(sAssignmentIId) should return the title for the dialog

    showPluginAssignmentDialog()

    Displays the dialog for assigning a single Plugin to a Property editor. When the dialog closes, the sap.dm.dme.podfoundation.control.PropertyEditor#updateAssignedPlugin function will be called passing the assigned plugin and the assignment ID that was passed to the constructor for this class.

    The PropertyEditor that invokes this method can implement the following functions: 

    getAssignmentPlugin() should return the current assigned plugin id
getPluginAssignmentDialogTitle(sAssignmentId) should return the title for the dialog
