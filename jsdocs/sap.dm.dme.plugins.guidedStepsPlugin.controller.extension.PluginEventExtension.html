PluginEventExtension

The PluginEventExtension allows modification of core plugin behaviour when processing Guided Step Plugin events. Custom extensions can be created to override or modify core behaviour. The following POD Plugin event functions can be overridden by custom extensions:

   onSelectionChangeEvent()
   onIconTabFilterSelectEvent()
   onNextButtonPressEvent()

new PluginEventExtension()

Constructor for the Plugin Event Extension.
