CreateExtension

CreateExtension

sap.dm.dme.plugins.worklistPlugin.controller.extension.CreateExtension provides functions that creates the list table and toolbar. Custom extensions can be created to override or modify core behaviour.

Constructor

new CreateExtension()

Constructor for the Worklist plugin Create Extension. The following functions can be overridden by custom extensions:

   createWorklistTable()
   enhanceWorklistTable()
   getTableConfiguration()
   getListConfiguration()
   getColumnConfiguration()
   createToolbar()
   loadWorklist()
   loadWorklistCount()

Extends

  • sap.ui.core.mvc.ControllerExtension

Methods

createWorklistTable(oListConfigurationData) → {object}

Creates a table based on input list configuration data. This function can be overriden by custom extensions as follows:

  For OverrideExecution.Instead:
     The oListConfigurationData object contains the list information that
     is defined in the configuration for the plugin.
     This function must return the Table data object.

     let oTableData = createWorklistTable: function(oListConfigurationData) {}

  For OverrideExecution.Before:
     The oConfigurationData object contains the information that will
     be used to create the table.

     createWorklistTable: function(oConfigurationData) {}

  For OverrideExecution.After:
     The oTableData object contains what is being returned (described below)

     createWorklistTable: function(oTableData) {}
Parameters:
Name Type Description
oListConfigurationData object

List configuration from DB
Throws:

Error if error occurs during table creation

Returns:

Object containing following data:

  {
       table: Table,
       columnListItem: ColumnListItem,
       listConfiguration: Object containing List Configuration data
  }
