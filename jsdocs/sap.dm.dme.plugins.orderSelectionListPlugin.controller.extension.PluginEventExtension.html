PluginEventExtension

PluginEventExtension

sap.dm.dme.plugins.orderSelectionListPlugin.controller.extension.PluginEventExtension provides functions that handles plugin events. Custom extensions can be created to override or modify core behaviour.

Constructor

new PluginEventExtension()

Constructor for the Order Selection List plugin Event Extension. The following functions can be overridden by custom extensions:

   onRefreshOrderList()
   onChangeResource()
   onChangeWorkCenter()
   onChangeMaterialNumber()
   onChangePlannedDateRange()
   onPodSelectionChangeEvent()
   onOrderStatusFilterChangeEvent()
   onSfcStatusFilterChangeEvent()
   onListPressEvent()
   onSelectionChangeEvent()
   onTableModelUpdateFinished()
